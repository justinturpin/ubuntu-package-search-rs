About

What is this?

This is just a search engine for Ubuntu Packages.

Why did you make this?

Because I feel like the official Ubuntu package search is kind of bad. I really like the Alpine Linux package search and wished that Ubuntu had something like it, so I made it. Also I wanted to learn Rust.

What is this made in?

Rust, using Gotham and Rusqlite.
